视觉助手任务：提取图中所有文本，并描述所有非文本元素（图标、符号、形状、物体等）。



\*\*要求：\*\*

1\.  对于文本元素：转录确切的文本内容。若内容为文档或公式，请使用 Markdown 和 LaTeX 格式。

2\.  对于非文本元素：提供简洁清晰的描述。

3\.  对于每个元素，请注明：

&#x20;   \*   `position`: 大致位置（如：左上角、中部、右下角）。

&#x20;   \*   `relationship`: 与邻近关键元素的空间关系（如：位于“标题”右侧）。

4\.  最后，提供一个基于视觉呈现的整体概览 (`Overview`)，描述图像结构与主题，不做解读。



\*\*输出格式：\*\*

```json

{

&#x20; "ImageAnalysis": {

&#x20;   "Elements": \[

&#x20;     {

&#x20;       "type": "text",

&#x20;       "content": "内容或描述",

&#x20;       "position": "位置",

&#x20;       "relationship": "空间关系"

&#x20;     }

&#x20;   ],

&#x20;   "Overview": "整体视觉内容与结构概览"

&#x20; }

}

```
